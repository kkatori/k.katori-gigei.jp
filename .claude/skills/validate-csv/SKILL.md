# validate-csv

生成されたCSVファイルをrakumoプロファイルフォーマット仕様と照合し、問題を報告する。

## 手順

1. `CSVフォーマット/rakumoフォーマットCSV.xlsx` を読み取り、正規のカラム定義（31カラム）を確認する
2. 引数で指定されたCSVファイル（`$ARGUMENTS`）を読み込む
3. 以下を検証する:
   - カラム数が31であること
   - ヘッダー名がフォーマット定義と一致すること
   - 必須フィールド（ユーザーID、メールアドレス等）が空でないこと
   - 日本語文字列のエンコーディングが正しいこと（文字化けがないこと）
   - 数値フィールド・日付フィールドの形式が妥当であること
4. 検証結果をサマリーとして出力する（OK件数、NG件数、エラー詳細）

## 出力形式

```
検証結果: [ファイル名]
- 総レコード数: N
- OK: N件
- NG: N件
  - 行X: [エラー内容]
  - 行Y: [エラー内容]
```
