# 初回メール自動返信 (Google Apps Script)

受信した初回メールに対して、件名・本文のキーワード条件に基づき自動返信を行うGoogle Apps Scriptです。

## 機能

- 件名または本文に指定キーワード（複数設定可、いずれか一致）が含まれるメールに自動返信
- 初回メールのみ対象（スレッド内の返信には反応しない）
- 返信の件名・本文は任意に設定可能
- Gmailラベルによる重複返信防止・処理済み管理
- Google Workspace の30分実行制限に対応（ページネーション+タイムリミット）

## ファイル構成

| ファイル | 内容 |
|----------|------|
| `autoReply.gs` | メインスクリプト（1ファイル完結） |

## 設定手順

### 1. GASプロジェクトの作成

1. [Google Apps Script](https://script.google.com/) にアクセス
2. 「新しいプロジェクト」を作成
3. `autoReply.gs` の内容をエディタに貼り付け
4. プロジェクト名を任意で設定（例：「初回メール自動返信」）

### 2. 設定値の編集

`autoReply.gs` 上部の定数を環境に合わせて編集します。

```javascript
// 自動返信メールの件名
const REPLY_SUBJECT = 'お問い合わせありがとうございます';

// 自動返信メールの本文
const REPLY_BODY = [
  'お問い合わせいただきありがとうございます。',
  '',
  '内容を確認の上、担当者より改めてご連絡いたします。',
  'しばらくお待ちくださいますようお願い申し上げます。',
  '',
  '※このメールは自動送信です。',
].join('\n');

// 自動返信の条件キーワード（いずれか一致で返信）
const MATCH_KEYWORDS = [
  'お問い合わせ',
  '資料請求',
  '健康診断',
];
```

### 3. トリガーの設置

1. GASエディタで関数セレクタから `setupTrigger` を選択
2. 「実行」をクリック
3. 初回実行時にGmailへのアクセス権限を承認

これで5分間隔の自動実行が開始されます。

### 4. 動作確認

1. キーワードを含むテストメールを対象アカウントに送信
2. 数分以内に自動返信が届くことを確認
3. Gmailで「自動返信/返信済み」ラベルが付与されていることを確認

## Gmailラベル

スクリプトは以下のラベルを自動作成・管理します。

| ラベル | 用途 |
|--------|------|
| `自動返信/確認済み` | チェック済みの全スレッドに付与（再チェック防止） |
| `自動返信/返信済み` | 自動返信を送信したスレッドに付与（運用確認用） |

## 停止方法

GASエディタで `removeTrigger` を実行するとトリガーが削除され、自動返信が停止します。

## 注意事項

- 1つのメールアドレスにつき1つのスクリプトプロジェクトとして運用してください
- Google Workspace環境では実行時間上限30分に対し、25分でタイムアウトするよう設定しています
- 送信失敗したメールはラベルが付与されず、次回実行時にリトライされます
